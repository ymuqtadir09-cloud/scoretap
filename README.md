# 🏏 GullyScore

A gully cricket scoring app built with React + Vite.

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Features
- Live ball-by-ball scoring
- Wide, No Ball, Bye, Leg Bye extras
- 2-innings match with target + RRR
- Over-by-over history
- Match history saved to localStorage
- Undo last ball
